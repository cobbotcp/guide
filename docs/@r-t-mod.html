r%t-mod — konfiguracja

Szybki poradnik: logowanie do panelu, zmiana hasła, utworzenie webhooka Discord i wklejenie danych do Build Fabric.

1. Zaloguj się do panelu

  1. Wejdź na stronę: cp.amnesia.zip (otworzy się w nowej karcie).
  2. Jeśli pojawi się ekran weryfikacji, kliknij Continue, a następnie zaloguj się na swoje konto.

2. Zmień hasło po zalogowaniu

  1. Po zalogowaniu przejdź do Profile & settings.
  2. Wybierz zmianę hasła, wpisz nowe hasło i zatwierdź.

3. Stwórz webhooka Discord

  1. Wejdź na swój serwer Discord.
  2. Utwórz kanał tekstowy, np. logs.
  3. Wejdź w edycję kanału i otwórz sekcję Integrations (tak jak na screenie discordwebhook.png).
  4. Kliknij Create Webhook.
  5. Skopiuj Webhook URL.

4. Wpisz dane w Build Fabric

  1. Otwórz menu Build Fabric.
  2. Wklej skopiowany Webhook URL w odpowiednie pole.
  3. Zapisz zmiany.

5. Logowanie przez sessionid

  1. Pobierz klienta Minecraft: litemclauncher.msi.
  2. Przejdź przez instalację.
  3. Wklej token w pole: Microsoft Session Login oraz Paste your Minecraft session token (JWT):.
  4. Kliknij Launch.

Jeśli webhook nie działa od razu, sprawdź czy URL został skopiowany w całości i czy kanał Discord nadal istnieje.
